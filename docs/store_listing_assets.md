# Chrome Web Store 详情页素材

您可以直接使用以下内容来填写 Chrome Web Store 的商店详情页。

## 详细说明 (Detailed Description)

**喜马拉雅下载器 (Ximalaya Downloader)**

一款专为喜马拉雅听书爱好者设计的强大工具。它可以让您轻松批量下载您已购买的音频内容，方便在各种设备上离线收听。

### 🚀 核心功能：

*   **下载全部**：一键开启自动化下载。插件会自动滚动页面、抓取所有音轨信息，并逐一进行下载。它具备智能去重功能，会自动跳过本地文件夹中已存在的文件。
*   **下载本页**：灵活下载。如果您只想下载当前页面的内容，这个按钮是您的最佳选择。
*   **本地列表 (智能去重)**：您可以先选择本地存放音频的文件夹，插件会扫描文件名。这样在批量下载时，已有的章节不会被重复下载，节省您的时间和带宽。

### 🛠️ 使用步骤：

1.  在喜马拉雅官网登录您的账号，并确保已购买相关音频。
2.  进入该音频的专辑详情页。
3.  点击页面右下角新出现的下载控制按钮。
4.  （推荐）先点击“本地列表”选择您的下载目录，然后点击“下载全部”开始无忧下载。

商店允许上传最多 5 张截图（推荐尺寸 1280x800）。建议按照以下顺序准备：

1.  **封面图**：展示插件在专辑页面的按钮布局。
2.  **功能演示**：一张展示点击“下载全部”时，按钮文字变为“正在下载...”状态的截图。
3.  **说明页面**：展示点击插件图标后弹出的介绍页面（即 `index.html` 的精美 UI）。
4.  **去重演示**：展示控制台或 UI 提示“已跳过重复文件”的画面。
